# Ida's Unisex Beauty Salon

Website for Ida's Unisex Beauty Salon, Parow, Cape Town.

## Deployment

This is a static HTML site. It is deployed via [Vercel](https://vercel.com).

To deploy:
1. Push this repo to GitHub
2. Import the repo in Vercel
3. Vercel auto-detects it as a static site — no build settings needed
4. Click **Deploy**

## Local Preview

Just open `index.html` in your browser — no server required.
